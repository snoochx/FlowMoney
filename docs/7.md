# Обоснование выбора стека технологий

Для разработки приложения **FlowMoney** был выбран следующий стек технологий:

- **Язык программирования:** Java  
  Обеспечивает стабильность, широкую поддержку Android-платформы и наличие большого количества библиотек.

- **Платформа:** Android  
  Позволяет создать мобильное приложение, доступное широкому кругу пользователей.

- **СУБД:** SQLite  
  Лёгкая встраиваемая база данных, оптимально подходящая для автономных мобильных приложений.

- **ORM:** Room  
  Упрощает работу с базой данных, обеспечивает типобезопасность и защиту от ошибок SQL.

- **Архитектура:** MVVM  
  Повышает читаемость кода и упрощает сопровождение проекта.

- **Инструменты разработки:** Android Studio, Gradle  
  Официальная среда разработки под Android с полной поддержкой необходимых инструментов.

Выбранный стек технологий соответствует требованиям автономности, надёжности и производительности системы.
